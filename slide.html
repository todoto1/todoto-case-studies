매주 발표 슬라이드를
노션에서 한 번 골라 14장 자동 완성

노션에 모아둔 자료가 슬라이드 14장에 자동으로 배치. 한 세션 만드는 데 6시간 걸리던 게 15분으로.

  • 노션 자료가 슬라이드 14장에 알아서 배치
  • 발표일·청중·목표가 한 번에 자동으로 채워짐
  • 발표 끝나면 슬라이드 주소가 노션에 자동으로 다시 적힘
  • 모바일로 미리 세션 정해두고, 컴퓨터에서 슬라이드만 뽑아내면 끝
전체 스토리 보기 →

매번 손으로 만들던 슬라이드 → 노션 한 번에 자동

매주 발표·웨비나·라이브를 하는 1인 사업자가, 슬라이드 14장을 매번 손으로 만들고·자료 복붙하고·발표 후엔 결과물이 흩어지던 흐름을, 노션에 자료만 모아두면 슬라이드·일정·다음 추천이 다 이어지는 시스템으로 바꾼 이야기.

01

왜 만들었나

매주 발표하는 1인 사업자가 겪던 문제

실제 운영 상황
"매주 발표·웨비나·라이브를 함. 블로그·강의록·인터뷰 자료는 노션에 다 있는데, 슬라이드로 옮기는 건 매번 처음부터 다시 — 디자인 맞추고·말투 맞추고·복붙에 30분, 발표 끝나면 결과물은 또 어디 갔는지 모름. 한 세션에 6시간씩. 청중도 '이거 들었는데 다음엔 뭐 들으면 좋지?' 가 막막."

대표 페인 포인트

  • 매번 슬라이드 14장을 손으로 — 디자인·글·말투 통일하느라 같은 작업을 반복
  • 노션 자료를 슬라이드로 옮기는 데만 30분 복붙 — 페이지마다 핵심 뽑고 다시 정리
  • 발표 후 결과물이 흩어짐 — 다음 세션이랑 안 이어지고, 시리즈로 묶어도 청중한테는 부담만
  • 모바일에선 기획이 안 됨 — 카페에서 떠오른 아이디어는 메모만 해두고 컴퓨터 앞에서 또 새로
  • 청중 입장에서 다음에 뭘 들어야 할지 모름 — 한 번 듣고 나가버림
02

어떻게 풀었나

실제 겪은 시행착오

1
워크샵용 패턴만 만들었더니 웨비나·라이브가 안 됨
처음에는워크샵용 슬라이드 패턴(목차·챕터·본문·요약·마무리)만 만들고 끝내려 했음. 그런데 웨비나·라이브·세미나는 결이 전혀 다름 — 청중을 끌어들이고 설득하는 흐름이 따로 필요
→
바꿨더니슬라이드 흐름을 14장으로 다시 짬 — 인사 → 후킹 → 고민 짚기 → 공감 → 질문 → 컨셉 → 증거 → 숫자 → 사례 → 비포·애프터 → 시연 → 보상 예고 → 전환 → 다음 추천. 발표가 15분이든 60분이든 알아서 분배해줌
2
"5주 시리즈" 식으로 묶었더니 청중이 부담
처음에는"SEO 마스터 5주" 식으로 1회차·2회차·3회차 묶어버림. 앞 회차 안 들으면 다음 이해가 어려워서, 중간에 합류하는 사람은 진입 자체가 부담
→
바꿨더니한 번에 하나씩 끝나는 '세션' 단위로 바꿈. 세션끼리는 "같이 들으면 좋은 세션" 연결만 둠. 청중은 관심 주제만 골라 듣고, 마지막 슬라이드에 다음 추천 세션이 자동으로 떠줌
3
노션 자료함에 정리된 글만 넣을 수 있다고 가정
처음에는노션 자료함에는 이미 잘 정리된 글만 들어간다고 가정. 그런데 인터뷰·대화·메모 같은 날것은 슬라이드로 못 가져옴 — 매번 손으로 정리해서 다시 넣어야
→
바꿨더니날것을 자료함에 넣기 전에 정제하는 단계를 따로 분리. 인터뷰·대화 → 핵심가치·페인·시행착오·비포애프터로 자동 구조화 → 자료함에 정돈된 페이지로 들어감. 이 정제 도구는 별도로 만들기로
4
미리보기가 정사각형으로 나오고 줄바꿈도 깨짐
처음에는대시보드 미리보기를 16:9로 맞추는 표준 방법을 썼는데 환경에 따라 비율이 정사각형 가까이로 깨짐. 안의 글자도 풀스크린이랑 줄바꿈이 달라서 미리보기랑 실제가 안 맞음
→
바꿨더니미리보기 박스 자체를 고정 크기로 두고 통째로 축소하는 방식으로 바꿈. 어디서 봐도 정확히 16:9, 실제 발표 화면이랑 줄바꿈도 똑같음
5
노션에 회차·시리즈명 필수로 했더니 운영이 안 맞음
처음에는노션 일정함에 "회차"·"시리즈명" 필드를 필수로 둠. 모든 세션이 시리즈 구조에 강제로 들어가서, 매주 다른 주제로 운영하면 빈 값만 쌓임
→
바꿨더니"회차"·"시리즈명" 빼버리고 "유형"(세션·워크샵·세미나·특강·라이브) + "난이도"(기초·중급·심화) + "다음 추천 세션" 으로 다시 설계. 이번 주는 SEO, 다음 주는 클로드 코드 — 매주 다른 주제로 가도 깔끔
03

실제 동작 화면

노션에서 세션 한 번 고르면 → 14장 완성 → 발표 → 노션에 자동 정리

새 슬라이드 만들기 — 노션에서 한 번 고르면 14장 완성

한 화면에서 — 종류·청중·자료·받고 싶은 행동·노션 세션

① 종류·테마(웨비나/워크샵/강의 × 기본/라이트) ② 발표 길이 ③ 제목·청중 ④ 청중에게 받고 싶은 행동(SNS 후기·뉴스레터·구매 등 6가지) ⑤ 단계별 보상(원하면 직접 쓰고, 비우면 AI가 알아서) ⑥ 참고 자료(노션 자료함에서 체크) ⑦ 📅 노션 일정함에서 세션 한 번 클릭 → ③~⑥이 한 번에 채워짐 + 다음 추천 세션도 자동으로 마지막 슬라이드에 들어감 → [슬라이드 생성] → 14장 완성 → 노션 일정함에 슬라이드 주소·상태가 자동으로 다시 적힘.
