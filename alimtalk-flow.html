Notion

발송 결과 자동 업데이트
영구 아카이브 저장
(1) AI 카피 생성·검수

Claude Code가 매번 카피 룰 외울 일을 없앴습니다

사용자는 어떤 발송 유형인지(워크샵·콘텐츠·마케팅·직접)만 고르면 됩니다. 5개 AI 에이전트가 알림톡·친구톡 8형식을 자동으로 구분하고 카피를 만듭니다. 광고 표현·글자수·변수 형식·머리말 규칙은 즉시 검사하고, AI 톤·의미 평가는 글이 흘러나오듯 점진적으로 표시됩니다.

[스크린샷 자리 — Claude Code 카피 생성·검수 화면]

핵심 동작

  • 5개 AI 에이전트 자동 선택 (알림톡·친구톡 자유형/템플릿형 + 검수 2종)
  • 광고·글자수·변수·머리말 즉시 검사 (수십 가지 카카오 규칙 자동)
  • AI 톤·의미 평가 스트리밍 (사용자 대기 시간 단축)
  • 발송 유형별 자유 필드 자동 세팅 (이름 = 변수 = 노션 컬럼명)
(2) 등록·발송

솔라피 콘솔에 들어갈 일이 없어요

검수 통과된 카피는 솔라피에 자동 등록되고 카카오에 검수 신청까지 자동으로 요청합니다. 발송 캠페인 모달에서 즉시·예약·자동 중 선택만 하면 됩니다. 노션에서 받는 사람을 필터로 고르고, 시점별 변수 매핑까지 한 모달 안에서 끝납니다.

[스크린샷 자리 — 솔라피 캠페인 5단계 모달]

핵심 동작

  • 솔라피 자동 등록 + 카카오 검수 자동 신청
  • 발송 캠페인 5단계 마법사 (템플릿·노션·필터·발송·콜백)
  • 즉시 발송 / 예약 / 자동(정기) 중 선택
  • 실패한 사람까지 결과 화면 한눈에
(3) 결과 정리

노션에 손으로 결과를 적을 일이 없어요

발송이 끝나면 노션의 row가 자동으로 업데이트됩니다. 발송 시각·결과·재시도 여부까지 사용자가 매핑한 컬럼에 따라 자동 정리. 풀 스냅샷이 영구 아카이브로 저장되어 다음 카피의 학습 데이터로도 활용됩니다.

[스크린샷 자리 — 노션 결과 자동 정리]

핵심 동작

  • 발송 성공·실패 row 자동 업데이트 (사용자 매핑 컬럼)
  • 영구 아카이브 저장 (output/complete/ 풀 스냅샷)
  • 다음 카피의 학습 데이터 1차 소스 (반려·승인 패턴)
