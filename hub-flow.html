(1) 본문·HTML 생성

Claude Code가 SEO 규칙·HTML 17블록을 한 번에

키워드와 주제만 입력하면 메인/가지 글 구조를 자동 결정하고, 제목·H2·서두 키워드 위치·메타 길이를 자동 적용합니다. 한 번 저장해둔 브랜드 톤·금지 단어가 모든 글 생성에 자동 주입되어 매번 같은 톤이 유지됩니다. 본문과 동시에 아임웹 HTML 17블록 패턴에 맞춰 자동 조립되어 코드를 손댈 일이 없습니다.

[스크린샷 자리 — Claude Code 글 작성·HTML 생성 화면]

핵심 동작

  • 메인/가지 글 구조 자동 결정 (분량·소제목 수·내부 링크)
  • SEO 규칙 자동 적용 (제목·H2·메타·키워드 위치)
  • HTML 17블록 자동 조립 (색·여백 고정)
  • 브랜드 톤·금지 단어 자동 주입
  • 이전 발행 글 톤 자동 참고 (자기강화)
(2) 자동 검수

색 14항목·금지 표현·톤을 자동으로

생성된 HTML을 자동으로 검수합니다. Chapter 번호 색·본문 줄간격·팁 박스 테두리 등 14항목 색·여백 규칙 + 금지 표현("알아보겠습니다" 등) + 톤 일관성을 자동 검사하고, 위반 시 빨강으로 표시합니다.

[스크린샷 자리 — 검수 리포트]

핵심 동작

  • 색·여백 14항목 자동 검사
  • 금지 표현 자동 검출 + 빨강 표시
  • 톤 일관성 평가 ("~합니다"/"~요" 분리)
  • 메타 디스크립션·슬러그 자동 추출
(3) 발행

아임웹에 복붙으로 끝

결과 HTML을 아임웹 본문에 그대로 붙여넣기만 하면 됩니다. 메타 디스크립션·URL 슬러그는 HTML 파일 맨 위에 일반 텍스트로 표시되어 아임웹 별도 입력 칸에 복붙. 외주 의뢰가 더 이상 필요 없어요.

[스크린샷 자리 — 아임웹 발행 화면]

핵심 동작

  • HTML 그대로 복붙 (아임웹 인라인 HTML 자리)
  • 메타·슬러그 텍스트 (HTML 주석 금지 — 아임웹 출력 함정 회피)
  • 외주 의뢰 불필요
Chapter 2.5

만들면서 배운 것 — 실제 시행착오

검색 노출·아임웹 HTML·톤·금지 표현 — 5가지 시행착오를 거쳐 지금 구조가 나왔어요.

1
한 글에 키워드 다 욱여넣음 — 오히려 검색 노출 약함
처음에는관련 키워드를 한 글에 다 넣으려다 분량 길어지고 키워드 밀도 떨어짐. 글끼리 연결도 평면적
→
바꿨더니메인 글과 가지 글로 분리 — 메인은 큰 키워드(긴 글), 가지는 세부 키워드(짧은 글). 메인은 가지에 링크, 가지는 메인에 다시 링크
2
SEO 규칙을 매번 머리로 — 일관성이 안 맞음
처음에는제목 키워드 위치·서두 200자 키워드·메타 길이 매번 검색해서 적용. 글마다 일관성 흔들림
→
바꿨더니이런 규칙을 AI에게 한 번 정리해서 알려줌. 다음부터 자동 적용. 매번 외울 필요 없음
3
아임웹 HTML — 색·여백을 손으로 매번 다름
처음에는아임웹 본문은 코드 그대로 받음. 글마다 챕터 번호 색·본문 줄간격이 미묘하게 달라짐
→
바꿨더니HTML 17개 블록 미리 만들어두고 텍스트만 갈아끼움. AI가 14항목 자동 검사
4
톤이 매번 흔들림 — 정보형·권유형 섞임
처음에는"~합니다"와 "~요"를 무작위 섞고 느낌표 남발. "알아보겠습니다" 같은 SEO 약화 표현도 자주
→
바꿨더니정보 전달은 "~합니다", 응원·공감은 "~요"로 분리. 금지 표현 목록 정리. AI가 위반 시 빨강 표시
5
메타 정보를 주석으로 적었다가 본문에 노출되는 사고
처음에는설명문·주소를 HTML 주석(`<!-- -->`)으로 적었더니 아임웹 게시 후 주석이 본문에 텍스트로 그대로 노출됨
→
바꿨더니주석 절대 금지로 규칙 추가. 메타 정보는 HTML 파일 맨 위에 일반 텍스트로

(2) 운영자 인터뷰

Q. 도입 전 가장 큰 어려움은?
보충 필요 예: "매번 키워드 조사부터 다시 시작해야 했어요. SEO 규칙도 매번 검색했고, 아임웹 HTML 색 검사하다 1편에 하루가 갔어요."
Q. 도입 후 가장 크게 달라진 점은?
보충 필요 예: "키워드만 입력하면 1시간 안에 발행 직전 글이 나와요. 검색 노출 잘 되는 구조도 자동이고, 아임웹 HTML도 그대로 복붙이라 코드 손댈 일이 없어요."
Q. 톤 일관성은 어떻게 유지되나요?
보충 필요 예: "정보 전달은 '~합니다', 응원·공감은 '~요'로 자동 분리되고, '알아보겠습니다' 같은 SEO 약화 표현은 위반 시 빨갛게 표시돼요."
Q. 어떤 사업자에게 추천하고 싶나요?
보충 필요 예: "자사몰 블로그 직접 운영하시는 분들. 외주 매월 N만원 줄이실 수 있고, 톤 일관성도 자동 유지돼요."
