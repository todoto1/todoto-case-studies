Instagram 운영을
노션 + AI 통합 OS로

PC가 꺼져 있어도 매시간 자동으로 데이터 수집. AI는 일반론 대신 우리 데이터 수치를 근거로 분석. 사용할수록 카피 품질이 올라가는 자기강화 SaaS.

  • PC 꺼져도 매시간 자동 수집 — 하이브리드 구조
  • "추측 금지" — AI 결론에 우리 데이터 수치를 반드시 인용
  • 잘 된 게시물의 패턴을 자동 학습 → 다음 카피에 반영
  • 모든 데이터를 본인 노션에 저장 — 락인 없음

인스타·노션·구글시트 → 한 대시보드

인스타 인사이트·노션·구글시트를 따로 보며 감으로 판단하던 운영자가, 매시간 자동 수집된 데이터 위에서 우리 수치 기반 인사이트로 의사결정하게 된 이야기.

01

왜 만들었나

1인 브랜드의 Instagram 운영 — 데이터 분산·휘발성·감에 의존

실제 운영 상황
"인스타 게시물 데이터는 일주일 지나면 사라지고, 팔로워 나이·지역 변화는 따로 추적이 안 됨. 노션·구글시트에 일부 옮겨도 매번 손이 가고. 그래서 결국 "이 콘텐츠가 잘 됐다"는 감으로만 판단하게 돼요. 어떤 패턴이 통하는지 데이터로 안 보임."

대표 페인 포인트

  • 인스타그램 게시물별 데이터가 일주일 후 사라짐 — 시계열로 추적 불가
  • 팔로워 나이·지역 변화는 따로 추적 안 됨
  • 100개 넘는 게시물인데 "어떤 패턴이 잘 됐는지" 안 보임
  • AI에게 분석 시키면 "스크롤 멈춤 유발" 같은 일반론만 — 우리 데이터 안 봄
  • PC가 꺼져 있으면 수집이 끊김

보충 필요 처음 만들기 시작한 구체적 계기·시기

02

어떻게 풀었나

실제 겪었던 시행착오

1
AI 분석이 일반론 — 우리 데이터를 안 봄
처음에는AI한테 "이 게시물 분석해줘" 했더니 "스크롤 멈춤 유발", "이목 집중" 같은 마케팅 격언만 반복. 우리 좋아요·조회수·저장 수치는 안 보고 일반론
→
바꿨더니AI에게 "추측 금지·우리 수치를 반드시 인용해라" 규칙을 강제로 넣음. 결과: "저장률 4.5%·평균 시청 22초·저장 35회" 식으로 우리 수치 인용. 주관 표현 0건
2
분석 규칙이 너무 많아 한 파일에 다 못 넣음
처음에는분석 규칙을 한 파일에 다 넣었더니 230줄 초과. 새 분석 모드 추가하기 점점 어려워짐
→
바꿨더니분석 규칙을 14개 모듈로 쪼개고, 분석 종류(주간·월간·연간·맞춤)에 따라 필요한 모듈만 자동 골라 조립
3
새 노션 형식이 우리 코드와 안 맞음
처음에는노션이 한 페이지 안에 여러 표를 묶는 새 형식을 도입했는데, 우리 코드가 옛 형식만 알아서 "지원 안 함" 거절. 브랜드 정보 연결이 안 됨
→
바꿨더니브랜드 정보 부분만 새 노션 형식 대응. 데이터 수집·발송 등 다른 부분은 옛 형식 그대로 — 안전하게 격리
4
자정 데이터 수집이 누락됨
처음에는5/12 0시 데이터가 비어 있음. 원인 3가지 — PC가 잠들어 있었고, GitHub 자동 시계가 분 단위로 부정확했고, 우리 코드가 정확히 0시일 때만 수집하도록 만들어져 있었음
→
바꿨더니"마지막 수집한 지 22시간 이상 지났으면 시간대 상관없이 자동 수집"으로 보정. 누락이 있으면 자동 복구
5
최근 50개만 갱신 — 옛 게시물 자료 사라짐
처음에는매일 최근 50개만 갱신. 30일 이상 된 게시물은 영상 주소가 만료되어 자막 추출 실패. 2023년 게시물부터 자료 사라짐
→
바꿨더니모든 게시물 매일 갱신(157개). 30일 이상은 통계는 skip하고 영상 주소만 갱신 — 만료 회피
6
노션 카드 결과 영역이 짧고 표도 깨짐
처음에는AI 기획서를 노션 카드의 짧은 메모 자리에 저장. 2000자 제한·표 렌더링 안 됨. 사용자 노션 카드 본문은 그대로 비어 있음
→
바꿨더니노션 카드 본문에 직접 저장. 제목·목록·표·구분선까지 정상 렌더링. "📝 todoto 기획서 — 시각" 헤더로 누적 추가

03

실제 동작 화면

노션 상태 변경 → 10초 후 본문에 기획서 자동 작성

Content Planning — 노션 트리거 + 자동 기획서

「콘텐츠 아이디어 DB」 상태를 "기획요청"으로 → todoto가 본인 브랜드 톤·잘 됐던 게시물 패턴 반영한 기획서 자동 작성

브랜드 정보(노션 DB 연결) + 잘 됐던 게시물 패턴(상위 25%) + 그 게시물의 캡션·자막·자막 외 텍스트 + 사용자가 입력한 주제·형식·길이·CTA → AI가 본문에 마크다운으로 누적 작성. "💡 근거"로 어떤 게시물을 참고했는지 표시.
