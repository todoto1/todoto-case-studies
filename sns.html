팔로워 추이·게시물 시계열·캡션·영상 자막·이미지 텍스트까지 — 모든 데이터를 본인 노션에 영구 저장하고 세분화해서 본다.

  • 게시물별 시간 흐름 시계열 — 좋아요·조회수·저장 변화 추적

  • 캡션·영상 자막·이미지 텍스트 자동 추출 — 콘텐츠 패턴 분석
  • 모든 데이터를 본인 노션에 영구 저장 — 락인 없음

데이터 휘발·표면적 → 세분화·영구

인스타가 보여주는 7일 평균 인사이트로는 안 보이는 패턴을, 게시물별·팔로워별·시간별로 세분화해서 보게 된 이야기.

01

왜 만들었나

인스타가 보여주는 데이터는 표면적·휘발성 — 진짜 패턴이 안 보임

실제 운영 상황
"인스타 인사이트는 지난 7일 평균만 보여줘요. 게시물 1개가 시간 흐름에 따라 어떻게 떴는지는 안 보이고, 팔로워 변화도 '오늘 몇 명'만. 100개 넘는 게시물인데 어떤 형식·길이·후크가 잘 됐는지 패턴이 안 보이고, 일주일 지나면 게시물 데이터가 또 사라져요."

대표 페인 포인트

  • 인스타 인사이트는 7일 평균만 보여줌 — 게시물별 시간 흐름은 추적 불가
  • 팔로워는 "오늘 몇 명"만 — 누적·증감·지역·연령 변화는 따로 안 보임
  • 게시물 100개+인데 "어떤 형식·길이·후크가 잘 됐는지" 패턴 안 보임
  • 캡션·영상 자막·이미지 안 텍스트가 따로 정리 안 됨 — 콘텐츠 분석 불가
  • 게시물 데이터가 일주일 지나면 사라짐 — 시계열 영구 보존 X

보충 필요 처음 만들기 시작한 구체적 계기·시기

02

어떻게 풀었나

실제 겪었던 시행착오

1
게시물 시계열을 어디에 저장할지 — 인스타는 안 알려줌
처음에는인스타가 게시물 시계열을 안 보여주니 매주 좋아요·조회수를 스크린샷으로 정리 → 너무 번거로워 결국 안 함. 시간 흐름 데이터가 없음
→
바꿨더니매시간 자동으로 게시물별 데이터를 노션에 추가 저장. 게시물 1개에 시간별 스냅샷이 쌓여서 "어제 대비·일주일 전 대비" 변화가 시계열 차트로 보임
2
PC가 꺼져 있으면 수집이 끊김
처음에는로컬 PC에서 스케줄러로 매시간 수집 → PC 잠들거나 끄면 데이터 누락. 외출·여행 후 며칠치 사라짐
→
바꿨더니GitHub의 자동 시계가 매시간 대신 수집. PC 꺼져 있어도 데이터는 노션에 쌓임. 인사이트 분석만 PC 켰을 때
3
자정 데이터 수집이 누락됨
처음에는5/12 0시 데이터가 비어 있음. 원인 3가지 — PC가 잠들어 있었고, GitHub 자동 시계가 분 단위로 부정확했고, 우리 코드가 정확히 0시일 때만 수집하도록 만들어져 있었음
→
바꿨더니"마지막 수집한 지 22시간 이상 지났으면 시간대 상관없이 자동 수집"으로 보정. 누락이 있으면 자동 복구
4
최근 50개만 갱신 — 옛 게시물의 영상·이미지가 만료
처음에는매일 최근 50개만 갱신. 30일 이상 된 게시물은 인스타가 주는 영상·이미지 주소가 만료되어 자막·텍스트 추출 실패. 2023년 게시물부터 자료 사라짐
→
바꿨더니모든 게시물 매일 갱신(157개). 30일 이상은 통계는 skip하고 영상·이미지 주소만 다시 받음 — 만료 회피
5
AI 분석이 일반론 — 우리 데이터를 안 봄
처음에는AI한테 "이 게시물 분석해줘" 했더니 "스크롤 멈춤 유발", "이목 집중" 같은 마케팅 격언만. 우리 좋아요·조회수·저장 수치는 안 보고 일반론
→
바꿨더니AI에게 "추측 금지·우리 수치를 반드시 인용해라" 규칙 강제. 결과: "저장률 4.5%·평균 시청 22초·저장 35회" 식으로 우리 수치 인용. 일반론 0건
6
잘 된 게시물 패턴을 찾으려면 100개를 일일이 봐야
처음에는"어떤 후크·길이·이모지·CTA가 잘 됐나" 보려면 게시물 100개를 일일이 비교 — 사실상 안 함
→
바꿨더니상위 25% 게시물의 공통 패턴을 자동 추출 — 후크 단어 빈도·평균 길이·이모지·CTA·발행 시점까지. 매주 자동 재계산

보충 필요 다른 시행착오가 있다면

03

실제 동작 화면

게시물별 시계열 + 팔로워 추이 — 세분화 분석

Posts — 게시물별 시간 흐름 + AI 채팅

게시물 1개 클릭 → 시간별 좋아요·조회수·저장 변화 차트 + 캡션·자막·이미지 텍스트까지

16가지 메트릭(좋아요·조회수·저장·공유·평균 시청·프로필 방문·팔로우 전환·참여율 등)이 표로. 게시물 클릭 시 시간 흐름 차트 + 같은 형식 다른 게시물과 비교 + AI 채팅(자동 컨텍스트로 메트릭·캡션·자막 다 전달). "다른 게시물과 비교해줘"·"카피 추천해줘" 같은 질문에 우리 수치 기반으로 답.

[스크린샷 자리 — Posts 모달 (시계열 차트 + AI 채팅)]
Growth — 팔로워 추이 + 데모그래픽 변화

팔로워 누적·증감 토글 + 지역·연령·성별 변화 + 30일 예측

팔로워 장기 추이를 "누적" 또는 "일별 증감"으로 토글. 증감 막대 +녹/-빨 + 값 라벨. 데모그래픽(지역·연령·성별) 스냅샷 시계열 + 선형회귀 30일 예측. "오늘 몇 명"이 아닌 "어디서·어떤 연령대가 늘었는지"가 보임.
